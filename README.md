《铁巢重炮》游戏弹道计算器/ 
Iron Nest Artillery Ballistic Calculator

作者/Author：
Alex-Symbiosis

描述/Description：
这是一个用于计算《铁巢重炮》游戏中弹道轨迹的工具，可离线使用，输入发射距离即可快速算出炮弹仰角，跳过游戏中繁琐的数值输入。
此功能简单，UI界面舒适，欢迎使用，未来将会随着游戏发展持续更新实用功能分支！

This is a tool designed to calculate ballistic trajectories for the game *Iron Nest*.
It can be used completely offline—simply input the firing distance to quickly determine the required elevation angle, 
bypassing tedious in-game calculations. Featuring a clean and user-friendly UI, it is ready for immediate use.
More practical features and branches will be continuously updated as the game evolves. Welcome to use it!
